# AI Content Research & Idea Engine
Live Link "https://ai-content-engine-bhbl6pswrj4h47gjh8bqmm.streamlit.app/"

## Overview
This project is an AI-powered automation system that collects trending AI-related news and generates content ideas automatically. It helps content creators and marketing teams reduce manual research time by automatically producing hooks, scripts, and content insights.

## Problem
Content teams spend significant time manually researching trending topics across multiple platforms. This process is repetitive and inefficient.

## Solution
The AI Content Research Engine automates the research process by:
- Collecting trending AI news from RSS feeds
- Generating content hooks and scripts automatically
- Displaying results in an interactive dashboard

## System Architecture

RSS Feeds  
↓  
Python Automation Script  
↓  
Content Idea Generation  
↓  
Streamlit Dashboard  

## Technologies Used
- Python
- Streamlit
- RSS Feed Parsing
- Pandas

## Features
- Automatically collects trending AI news
- Generates content hooks
- Generates short scripts for content creation
- Provides a simple dashboard interface
- Helps creators identify viral content ideas quickly
